Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for its child elements using event bubbling, improving performance and reducing memory usage.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism where an event starts from the target element and propagates up to its ancestors in the DOM hierarchy.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create an element using document.createElement(), than add with appendChild() and remove it with removeChild().
